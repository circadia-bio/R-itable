# Ritable colour palette

A named character vector of the four brand colours used throughout
Ritable figures and documentation.

## Usage

``` r
Ritable_colours
```

## Details

- `pink`:

  `#FE9EC7` – primary accent; significant traits, sex effect

- `blue`:

  `#44ACFF` – primary brand colour; estimates, CI lines

- `sky`:

  `#89D4FF` – secondary blue; muted elements, CI shading

- `cream`:

  `#F9F6C4` – soft highlight; code backgrounds

## Examples

``` r
Ritable_colours
#>      pink      blue       sky     cream 
#> "#FE9EC7" "#44ACFF" "#89D4FF" "#F9F6C4" 
```
